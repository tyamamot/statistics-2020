## 講義目的・到達目標（シラバスより）
データに基づいて現象を客観的に理解し，推測することは現代社会において必要不可欠ともいえる技術である．本講義では，そのための方法論である統計および確率の基礎について学ぶ． 本講義の到達目標は，統計および確率の基礎理論を修得し，「統計的なものの見方」の基礎を身につけることである．
